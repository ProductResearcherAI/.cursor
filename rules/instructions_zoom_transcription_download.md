# Инструкция по скачиванию транскрибаций из Zoom и загрузке в панель проекта

## Шаг 1: Скачивание транскрибации из Zoom

### Для облачных записей Zoom:

1. **Войдите в Zoom Web Portal**
   - Откройте браузер и перейдите на [zoom.us](https://zoom.us)
   - Войдите в аккаунт Zoom по логину и паролю - interviewkotilev@gmail.com, interview54*

2. **Найдите запись интервью**
   - В центре выберите "Открытые записи"
   - Найдите нужную запись интервью по дате или названию
   - Нажмите на название записи

3. **Скачайте транскрибацию**
   - Нажмите на иконку скачивания и в скачанных файлах найдите раздел "Transcript" (Транскрибация)
   - Файл будет скачан в формате .txt или .vtt

Перетащите загруженную транскрибацию в панель проекта с файлами и выполните команду git push.
