## 31.05.2025 - CK

-   **Правило добавлено:** `csv_consolidation_rule.mdc`
-   **Назначение:** Определяет строгие условия и итерационный процесс для создания и обновления CSV-таблиц из наборов данных. Внесено для предотвращения ошибок "съездов" данных, обеспечивая их корректную обработку при создании CSV. 
Включает правила обработки заголовков, данных батчами, обрамления ячеек двойными кавычками, использования одинарных кавычек для внутреннего цитирования, обработки пустых значений и обязательной проверки релевантности данных столбцам после каждой итерации.
-   **Автор изменения:** CK
